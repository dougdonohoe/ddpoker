DD Poker - Texas Hold'em - Official Site
DD Poker is a full-featured open-source Texas Hold'em (no-limit, pot-limit, limit) simulator, allowing you to play against AI and human opponents.
  • Free and fully functional poker entertainment software.
  • Play online with your friends in private games.
  • Practice your no-limit, pot-limit and limit skills against advanced computer opponents.
  • Design and run your own tournaments.
  • Full details...
It also has a poker clock for running your own home tournaments and a sophisticated calculator tool for exploring the strength of various hands and the odds of winning.
You can play against other human opponents by setting up an online game and sharing the game URL (unless everyone is running on the LAN, this requires opening a port to allow inbound connections to your computer).
There used to be an Online Portal that facilitated finding games posted by other users, tracked player profiles, and maintained tournament histories. We no longer run this server, but you can run your own using the DD Poker source code.
Installers are built courtesy of ej-technologies' multi-platform installer builder, Install4j.
Although DD Poker is free software, we still incur real expenses for ongoing development. If you enjoy DD Poker, please consider donating to help defray these costs.
