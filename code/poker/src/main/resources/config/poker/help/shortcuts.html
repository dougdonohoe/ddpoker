Chat, Mouse and Keyboard Shortcuts   

Chat Shortcuts
CTRL-L
CMD-L
 Open Online Lobby Opens the online lobby chat window from any location from within DDPoker.
/8s Displays Card When typed in any chat window, the card represented after the forward slash "/" will be printed in graphics.
Example: Type /8s9s to display a .
Use s for spades, h hearts, d diamonds, c clubs, T or 0 or 10 for 10, j jack, q queen, k king, a ace.
Mouse Double Click Select Word Double click any word in the chat to select the entire word you wish to copy to the clipboard by then using CTRL-C.
CTRL-C
CMD-C		 Copy Chat To copy a section of chat, highlight the desired section with the mouse. To highlight, place the mouse cursor at the start of the selection, press and hold the left mouse button as you drag the mouse over the desired section and release the mouse button. While the section is highlighted, press CTRL-C on Windows or CMD-C on mac to copy the selection to the clipboard.
CTRL-A
CMD-A		 Copy All Chat Click anywhere in the chat text and press CTRL-A or CMD-A to copy the entire chat dialog to the clipboard.
CTRL-V
CMD-V		 Paste Chat Left click the mouse cursor in the chat entry field or other application where you wish to paste the contents of the clipboard.
Press CTRL-V on Windows or CMD-V on mac to paste.

Right Click 		Pop-up Menu Export: To export the last 500 lines of chat, right click anywhere in the chat text, then click export. Choose a location to save the chat.html file to. By double clicking the chat.html file you can automatically view the chat text with your web browser.

Player Info: Right click a player name to choose the player info menu item to displays a player's aliases.

Mute / Un-mute Right click a player name to mute or un-mute a player's chat. It will automaticlly add or remove players to the muted player list.

Ban / Un-ban Right click a player name to ban or un-ban a player's chat. It will automaticlly add or remove players to the banned player list for future games they host.

Alt Allows Action in Chat If the focus is in the in-game chat window, you may press the Alt key before any of the action keys to act. Example: You are typing a chat message, you may press Alt - C to call if it is your turn to act. This function does not work in the online lobby chat. Hint: You may wish to disable the Move Window to Front option to prevent an accidental action in the game.

CTRL-U
CMD-U 		Open UDP Connection Status Screen Opens the UDP Connection Status Screen to view real time communication data.

This troubleshooting screen can help to determine communications issues. The UDP communications displays the Chat Server when you have the online chat lobby open and your connection to a host computer if you have joined an online game. If you are hosting, all your guests connection status will also be displayed. This screen will not display connection status of the old TCP/IP game connections.

Export: You can right click the UDP Connection Status Screen to Export the statistics to a .csv file which can be read by Microsoft Excel. Hint: You may need to expand the size of the connection screen window in order to complete the export process.

Note: In order to host a game using the new UDP communications instead of the default TCP/IP protocol, check the Use UDP checkbox in the online tab of your options screen. If you have security software you will need to configure it to allow DD Poker all TCP and UDP communications from port 11885 to 11889. If you have a router you will need to Port Forward on both TCP and UDP from port 11885 to 11889 to the local IP address of your host computer.

UDP Connection Displays the DD Poker Chat Server and/or other DD Poker user(s) you are connected with using UDP. If you have joined an online game you will see the name of your host. If you are hosting, you will see the name of each guest that has joined your game.
Remote IP:Port Displays the public IP address of the specified UDP Connection, and the UDP port used. If you see a port number other then 11885-11889, this the the NAT network address translation by your router to a second computer behind your firewall. If you are connected to another computer behind your firewall, the IP address shown is the Local IP address of that computer assigned in most cases by your router.
MTU Maximum Transmission Unit - This is the largest packet size that can be transferred between you and the chat server, or you and the specified user. When the connection is created, 576 is used and tests are made to determine how large a packet can be transferred up to 1500.
Time The total uninterrupted time connected to the chat server or specified user in Days, Hours, Minutes, and Seconds.
Ping The round trip time in milliseconds of a packet of data to be sent from your computer to the chat server or specified user and back. This figure is a rolling average over the last 100 messages.
Send Number of data packets sent to the chat server or specified user. Data packets can be game data or a chat message.
Resend Number of data packets re-sent to the chat server or specified user if a data packet did not make it and was resent. It is normal for a small percentage of UDP packets to be lost and resent when compared to the total number of packets sent.
Rcvd Number of data packets you received from the chat server or specified user.
Dups Number of data packets sent twice from the chat server or specified user if they did not receive an acknowledgement of the original message from you. It is normal for a small percentage of UDP packets to not return an acknowledgment and resend the message when compared to the total number of packets received.
Bytes-In Number of UDP data bytes received from the chat server or specified user. This data should continously increase as the connection is kept alive.
Bytes-Out Number of UDP data bytes sent to the chat server or specified user. This data should continously increase as the connection is kept alive.
Status Bar
Bytes 		The bytes in and bytes out displayed on the bottom status bar indicates the total current throughput of data bytes in and out of DD Poker at any particular moment. The number is a rolling average from the previous ten seconds. The high is the highest within that 10 second rolling window. The peak is the highest throughput in and out since DD Poker was launched.
